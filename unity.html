Unity

Tijdens deze workshop leerde we de basics van het programma Unity. Dit is een programma waarin je games kan programeren. we kregen een stappenplan dat we moesten volgen en we zouden dan eindigen met een extreem simpele versie van een game.





Stap 1 Terrain

de eerste opdracht was het maken van de grond van een level. nadat je deze ingeladen had. kon je met je muis bergen maken

Stap 2 FPS controller

hier moesten we een camara in brengen, dit wass je start plek wanneer je de game starten

Stap 3 Object

hierin moest je een simpele vorm(shape) in de game plaatsen.

Stap 4 Auto

het zelfde als de voorgaande opdracht alleen nu moest je de "shape" of te wel de auto eerst downloaden voor je hem erin kon plaatsen

Stap 5 Trigger

hier moesten we een event( het vuur) triggeren op een afstand. dit was een stuk gecompliceerder dan verwacht maar het is gelukt.

Stap 6 Exporteren

hier moesten we de game exporteren zodat het ook werkte als je geen unity had.

Mapmapper

Mapmapper is een projectie tool die je projecties kan laten manipuleren op een manier die jij wilt.

Bij deze mapmapper opgave was het de bedoeling om de geanimeerde kubus als een perfecte projectie te projecteren om 2 verschillende vlakken.
